Blockchain Visualization

Context: Information Visualization                         Duration : 3 weeks                        Team-members: Harsh Dev, Abhinav Garg

Problem Statement:
With the advent of Blockchain, my team and I decided to come up with a visual representation of what exactly blockchain is and how bitcoins are mined.

Tools used:
Javascript D3, Python, HTML,CSS

Link: Blockchain Viz

Research:
After careful analysis, we found that Blockchain and in particular bitcoin were the most talked about subjects in the world. We then decided to create a simple visualization that helps a layman understand what blockchain is and how bitcoins are mined.



Process:
We first understood what exactly blockchain is and how are blocks formed. My team and I wrote a python script to generate our own blockchain and have a hash value associated with each block. We further investigated as to how a transaction is approved within a network. The last piece of analysis was understanding different processors used to mine bitcoins. The speed, computational power and resource utilization were the key components that we understood in-order to prepare a visualization that represents different processors used to mine bitcoins

The basic aim of this entire project was to be able to elaborate as to how would you explain blockchain to your Grand-Ma?
